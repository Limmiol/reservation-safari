🔐 SECURITY SETUP
═══════════════════════════════════════════════════════════════

Before deploying to production, ensure:

✅ Generate secure secrets:
   openssl rand -base64 32    # JWT_SECRET
   openssl rand -base64 16    # WEBHOOK_SECRET

✅ Configure environment variables:
   NODE_ENV=production
   JWT_SECRET=<your-random-key>
   WEBHOOK_SECRET=<your-random-key>
   ALLOWED_ORIGINS=https://yourdomain.com

✅ Setup HTTPS/SSL:
   Use Let's Encrypt (free) or your provider's SSL

✅ Configure firewall:
   ufw allow 80    # HTTP
   ufw allow 443   # HTTPS
   ufw deny 3000   # Block direct access
   ufw deny 3001   # Block direct access

✅ Backup procedures:
   Regular backups of server/data.json
   Regular backups of server/uploads/
